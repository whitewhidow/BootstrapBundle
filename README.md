BootstrapBundle
===============



Will include jquery, bootstrap and font-awsome.


Installation:
-------------

```
"repositories": [
        {
            "url": "https://github.com/whitewhidow/BootstrapBundle.git",
            "type": "git"
        }
],
"require": {
        "whitewhidow/BootstrapBundle": "dev-master"
}
```


Usage in twig template:
-----------------------

```
{{ insertBootstrap() }}
```
